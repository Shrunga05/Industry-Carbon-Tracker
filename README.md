# Industry Carbon Tracker

## Description
Industry Carbon Tracker is a web-based application that helps industries estimate and monitor carbon emissions based on industrial activities and resource consumption.

## Features
- Carbon emission calculation
- Industry-wise tracking
- User-friendly interface
- Emission monitoring dashboard
- Environmental awareness support

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js

## Project Structure
- frontend/
- img/
- server.js
